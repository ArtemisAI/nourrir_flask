Les Coulisses de NourrIR

Bienvenue dans les coulisses de la création de NourrIR ! Ce projet a été développé avec passion et innovation dans le cadre du cours REI 1850 – Gestion des ressources humaines et intelligence artificielle à l'Université de Montréal (Session été 2025).

Notre Mission dans ce Projet

L'objectif était de concevoir et de prototyper une application web innovante explorant le potentiel de l'intelligence artificielle pour le bien-être. NourrIR est le fruit de cette exploration, visant à offrir une expérience utilisateur engageante et personnalisée.

Les Outils d'IA de Notre Créativité

Pour donner vie à NourrIR, nous avons utilisé une gamme diversifiée d'outils d'intelligence artificielle :

  • 🔎Recherche et Sources :
    • ChatGPT Deep Research : Recherche de sources et validation des informations.
  • 💡Idéation et Conception :
    • GPT-4 et modèles similaires : Développement conceptuel et structuration des idées.
    • Outils de génération d'images IA (type DALL-E/Midjourney via GPT-4o) : Génération de logos et d'illustrations.
    • Modèles de langage avancés : Élaboration des bases de la politique d'intégration.
  • 🎬Contenu Multimédia :
    • Suno : Composition musicale pour l'identité sonore de l'entreprise.
    • Synthesia.io et outils similaires : Création de vidéos explicatives animées.
  • 💻Développement et Débogage :
    • Assistants de code IA (ex: OpenAI Codex, GitHub Copilot) : Assistance à la génération de code.
    • Outils d'analyse de code assistés par IA : Pour le débogage et l'optimisation du code.
  • 🤖Chatbot RH :
    • n8n : Orchestration du chatbot, intégrant un système RAG (Retrieval Augmented Generation) utilisant une base de données vectorielle Pinecone pour extraire les informations internes.
  • 📊Présentations :
    • Canva : Conception des supports pédagogiques et présentations.

Workflow n8n du Chatbot RH

Cette approche visuelle avec n8n, combinée au RAG, offre plusieurs avantages : elle permet d'orchestrer facilement les appels API, d'intégrer et d'interroger nos données internes pour des réponses cohérentes et pertinentes, et de maintenir ou d'adapter le workflow sans écrire de code.

Pour toute question, contactez-nous.

Consultez le code source sur GitHub : https://github.com/ArtemisAI/nourrir_flask.

Cette application web fonctionne dans un conteneur Docker et est déployée sur Render.com. Elle est fournie à des fins pédagogiques uniquement.

Un Projet d'Apprentissage Continu

Ce projet représente une étape clé dans notre parcours d'apprentissage des technologies émergentes. Chaque outil et chaque phase de développement nous ont permis d'acquérir de nouvelles compétences et de mieux comprendre les enjeux et les possibilités de l'IA.
